# INTERBANK - Challengue Data Science

Solving a challenge for the Data Scientist role

## Enviroment

For environment creation, in command line execute:

```
py -3.11 -m venv ibk
```

To activate this environment, use:
```
ibk\Scripts\actívate
```

To install required libraries:

```
pip install --upgrade pip
pip install -r requirements.txt
```

## Folders

    ├── dataset        <- Data input for model
    
    ├── environment    <- Environment file for used libraries

## Challenge resolution

The challenge was solved using jupyter notebooks locally, but it can be executed in any environment with "Jupyter Notebook" using the environment defined in "requirements.txt".
